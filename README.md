# Anecdot

A modern blogging platform with user authentication, profile management, and theme customization.

## Tech Stack

**Frontend:**

- React.js
- Tailwind CSS
- Framer Motion
- Context API (State Management)

**Backend:**

- Node.js
- Next.js
- MongoDB (Database)
- Redis (Caching)

**Authentication:**

- JSON Web Tokens (JWT)
